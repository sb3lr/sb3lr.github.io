# WPScan

أداة لا غنى عنها لفحص أمن مواقع الووردبريس، تتيح لك اكتشاف القوالب والإضافات المصابة بثغرات.

### أمر الفحص الأساسي:
```bash
wpscan --url https://example.com --enumerate u
```
