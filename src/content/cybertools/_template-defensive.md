## بطاقة الأداة

- **الفئة**: Defensive
- **نوع الحماية**: Detection | Prevention | Response
- **البيئة المناسبة**: Endpoint | Network | Cloud

---

## نظرة عامة

اكتب تعريفًا واضحًا يشرح:

- نوع التهديدات التي تتعامل معها الأداة.
- الفرق بينها وبين البدائل.
- أهم بيئة تشغيل لها.

---

## طبقات الحماية التي تقدمها

| الطبقة | الوصف |
|--------|-------|
| **الاكتشاف** | كيف ترصد السلوك المشبوه |
| **المنع** | كيف توقف التهديد مبكرًا |
| **الاستجابة** | كيف تساعدك على الاحتواء والتحليل |

---

## إعداد سريع

### 1) التثبيت
```bash
# أوامر التثبيت (مثال)
apt install tool-name
```

### 2) الإعداد الأساسي
```bash
# تجهيز الإعدادات الأولية
tool-name --config setup
```

### 3) التشغيل والمراقبة
```bash
# تشغيل الخدمة ومتابعة السجلات
tool-name --start --monitor
```

---

## سيناريوهات دفاعية شائعة

- **التهديد 1**: طريقة الرصد.
- **التهديد 2**: آلية المنع.
- **التهديد 3**: خطة الاستجابة.

---

## مراقبة الأداء

| المؤشر | القيمة المقترحة | لماذا يهم |
|--------|------------------|-----------|
| **Detection Rate** | `%` مرتفع | لرفع كفاءة الرصد |
| **False Positives** | منخفض | لتقليل الإنذارات الخاطئة |
| **Response Time** | سريع | لتقليل أثر الحادث |
