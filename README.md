# 0xsb3lr Virtual OS Portfolio 🚀

مرحباً بك في مستودع مشروع **0xsb3lr Portfolio**، منصة عرض إلكترونية مصممة بهوية "نظام تشغيل افتراضي" (Virtual OS) مخصصة لمحترفي الأمن السيبراني والمطورين.

---

## ✨ المميزات (Features)

- **هوية بصرية فريدة**: تصميم عصري يعتمد على الـ "Glassmorphism" وتأثيرات الأنظمة التقنية المتقدمة.
- **إدارة المحتوى عبر Markdown**: سهولة فائقة في إضافة وتحديث الأدوات، الشروحات، والإنجازات عبر ملفات `.md`.
- **قسم الأدوات (Cyber Tools)**: استعراض للأدوات الهجومية والدفاعية مع نماذج جاهزة للإضافة.
- **قسم الشروحات (Tutorials)**: مكتبة شروحات تقنية منظمة ومنسقة بشكل احترافي.
- **Hall of Fame (الإنجازات)**: معرض للإنجازات والنجاحات الشخصية مع واجهات عرض سينمائية.
- **خريطة التهديدات (Threat Map)**: لوحة تفاعلية تحاكي مراقبة التهديدات السيبرانية الحية.
- **نظام الإنجازات (Gamification)**: نظام "Unlock" تفاعلي للمستخدمين أثناء تصفح الموقع.
- **ثنائي اللغة (i18n)**: دعم كامل للغتين العربية والإنجليزية مع تبديل سلس.

## �️ التقنيات المستخدمة (Tech Stack)

- **Framework**: [Astro 5.0](https://astro.build/) (للسرعة والأداء العالي).
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (للتصاميم المخصصة وسهولة الصيانة).
- **Components**: [React 19](https://react.dev/) (للعناصر التفاعلية المتقدمة).
- **Icons**: [Lucide React](https://lucide.dev/).
- **Content**: [Astro Content Collections](https://docs.astro.build/en/guides/content-collections/).

## 🚀 تشغيل المشروع محلياً (Setup)

تأكد من وجود **Node.js** مثبت في جهازك، ثم اتبع الخطوات التالية:

1. **تثبيت المكتبات**:
   ```bash
   npm install
   ```

2. **تشغيل بيئة التطوير**:
   ```bash
   npm run dev
   ```

3. **بناء النسخة النهائية**:
   ```bash
   npm run build
   ```
